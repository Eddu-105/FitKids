## Descripción del proyecto
FitKids es una plataforma web orientada a la promoción de hábitos saludables en niños, desarrollada como un sistema de información que integra funcionalidades de visualización, interacción y consumo de servicios. El trabajo fue inspirado en una empresa real perteneciente a un familiar, y aproximadamente el 90% de los textos del sitio fueron generados a partir de los requerimientos y pedidos directos de dicho familiar. El proyecto presenta contenido estructurado, gestión de datos mediante una API y una interfaz web moderna, accesible y responsiva, combinando frontend y backend para ofrecer una experiencia digital completa.
